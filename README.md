CryptoFont 0.0.1
================

A sleek webfont for your favourite cryptocurrencies.

## Getting Started

```html
<i class="cf cf-btc"></i> Bitcoin
```

**Get started at [AlexanderPoellmann.com/CryptoFont](https://AlexanderPoellmann.com/CryptoFont)!**
